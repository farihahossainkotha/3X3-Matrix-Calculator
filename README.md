# 3X3-Matrix-Calculator
A C++ matrix calculator for performing major 3X3 marrrix operations.

## Features
- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Determinant
- Matrix Transpose
- Matrix Inverse

## Technologies used
- C++
- Arrays
- Loops
- Conditional statement

## Future Improvements
-Support for nxn matrices
-Better UI formatting
-GUI version using Python
